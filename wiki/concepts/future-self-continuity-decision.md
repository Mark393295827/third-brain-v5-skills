# 未来自我连续性决策

> 未来自我连续性决策是通过让“未来的我”变得更具体、更像当前自我，来抵消时间折扣并改善长期选择的提问方法。
> (Source: [[src-20260522-decision-questions-science#^ki-future-self]]) WARNING: Single source

## Core Mechanism

```text
未来自我显像
  -> 情感连接增强
  -> 未来收益权重上升
  -> 当下短视冲动下降
  -> 更偏长期的选择
```

用户笔记将“未来的我会感谢什么？”解释为对未来自我连续性的操作化：不是抽象地要求长期主义，而是把未来自我当作与当前自我连续的真实行动对象。 (Source: [[src-20260522-decision-questions-science#^ki-future-self]]) WARNING: Single source

## Classifications / Comparisons

| Prompt | Bias targeted | Decision use |
|---|---|---|
| 未来的我会感谢什么？ | 时间折扣 | 长期后悔最小化 |
| 现在做什么最舒服？ | 即时满足 | 短期情绪调节 |
| 二十年后的我会怎么看？ | 身份连续性 | 高影响人生决策 |

## Implications / Applications

- 在重大选择前先回答未来自我问题，可以把注意力从即时收益转向长期后果。 (Source: [[src-20260522-decision-questions-science#^ki-future-self]]) WARNING: Single source
- 该问题适合作为 [[seven-question-decision-operating-system|七问决策操作系统]] 的第一问，因为它先确定时间尺度，再进入资源和执行判断。 (Source: [[src-20260522-decision-questions-science#^src-decision-questions-summary]]) WARNING: Single source
