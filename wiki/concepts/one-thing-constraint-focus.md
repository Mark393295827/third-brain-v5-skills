# 只做一件事约束

> 只做一件事约束用一个极端限制迫使决策者删除大多数合理选项，暴露真正高杠杆的下一步。
> (Source: [[src-20260522-decision-questions-science#^ki-one-thing]]) WARNING: Single source

## Core Mechanism

```text
选项过多
  -> 加入极端约束
  -> 删除低杠杆选项
  -> 聚焦唯一动作
  -> 减少决策瘫痪
```

用户笔记把“如果只能做一件事，我会选什么？”解释为约束促进创造力和聚焦的操作化，它通过人为收紧选择空间来提高优先级清晰度。 (Source: [[src-20260522-decision-questions-science#^ki-one-thing]]) WARNING: Single source

## Classifications / Comparisons

| Constraint | What it removes | What it reveals |
|---|---|---|
| 只能做一件事 | 伪优先级、并行幻觉 | 单一高杠杆动作 |
| 只能两周交付 | 过大范围 | MVP |
| 只能五分钟决定 | 过度分析 | enough-to-act 判断 |

## Implications / Applications

- 当任务列表过长时，先用只做一件事约束筛选，再用 [[timeboxed-experiment-decision-loop|时间盒实验决策循环]] 定义交付和指标。 (Source: [[src-20260522-decision-questions-science#^ki-one-thing]], [[src-20260522-decision-questions-science#^ki-two-week-ship]]) WARNING: Single source
- 该约束与 [[future-self-continuity-decision|未来自我连续性决策]] 结合时，可以把“未来会感谢什么”压缩为今天要执行的一个动作。 (Source: [[src-20260522-decision-questions-science#^ki-future-self]], [[src-20260522-decision-questions-science#^ki-one-thing]]) WARNING: Single source
