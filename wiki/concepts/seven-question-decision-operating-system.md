# 七问决策操作系统

> 七问决策操作系统是一套把长期视角、风险预算、短周期交付、实验验证、二阶后果、单一焦点和快速行动压缩到一次决策中的个人操作协议。
> (Source: [[src-20260522-decision-questions-science#^src-decision-questions-summary]]) WARNING: Single source

## Core Mechanism

```text
未来自我
  -> 降低短视
  -> 探索/利用标记
  -> 两周交付边界
  -> 自动化实验
  -> 成功副作用预演
  -> 单一高杠杆动作
  -> 五分钟内进入执行
```

该框架的核心机制是把模糊决策拆成七个固定检查点，每个检查点都对应一种认知或组织机制：未来自我连续性、探索/利用、时间盒、验证性学习、二阶后果、约束聚焦和决策时间盒。 (Source: [[src-20260522-decision-questions-science#^src-decision-questions-summary]]) WARNING: Single source

## Implications / Applications

- 对个人项目，先问“两周内一个人能否发货”，再问“能否变成自动化实验”，可以把大愿景压成最小可验证动作。 (Source: [[src-20260522-decision-questions-science#^ki-two-week-ship]], [[src-20260522-decision-questions-science#^ki-automated-experiment]]) WARNING: Single source
- 对战略选择，先标记探索/利用，再问成功是否制造系统性压力，可以避免把高不确定性赌注误当成例行任务处理。 (Source: [[src-20260522-decision-questions-science#^ki-exploration-budget]], [[src-20260522-decision-questions-science#^ki-success-disaster]]) WARNING: Single source
- 对拖延场景，用“只能做一件事”和“五分钟内决定”形成行动启动器，减少选择过载和过度分析。 (Source: [[src-20260522-decision-questions-science#^ki-one-thing]], [[src-20260522-decision-questions-science#^ki-five-minute-decision]]) WARNING: Single source
