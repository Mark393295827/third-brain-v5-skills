# One-Click Test Prompt

Paste this into Claude Code, Codex CLI, Gemini CLI, Cursor, or Windsurf after installing the skills.

```text
Use wiki-ingest on this mini source.

Source title: The Third Brain Loop
Source text:
Knowledge management is not enough when notes never change behavior. A useful cognitive system should turn input into structured understanding, then into behavior experiments, creative output, and a feedback loop. The value comes from repeated compounding: every source updates the wiki, every action updates the model, and every review improves the next cycle.

Create:
1. an immutable source note
2. one concept page for "Third Brain Loop"
3. one output note summarizing how this can be used in a daily workflow
4. a verification summary listing created files and single-source claims
```

## Expected 3-Minute Result

- One source note under `sources/`.
- One concept note under `wiki/concepts/`.
- One reusable output under `wiki/outputs/`.
- A verification summary that lists files and flags single-source claims.
