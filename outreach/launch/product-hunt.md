# Product Hunt Draft

## Name

Third Brain V5 Skills

## Tagline

Persistent Knowledge OS for AI coding agents

## Short Description

17 open-source Agent Skills for Claude Code, Codex, Gemini, Cursor, and Windsurf. Capture sources into Obsidian, reduce hallucinated completion claims with verification gates, manage context cost, and orchestrate multi-agent workflows.

## First Comment

```text
Hi Product Hunt,

I built Third Brain V5 Skills because AI coding agents are getting stronger, but most workflows still lose valuable knowledge at the end of each chat.

This project gives agents a durable workflow layer:

- Obsidian-backed LLM Wiki ingestion
- source-linked concept pages
- verification-before-claim discipline
- context and token budgeting
- agent workflow / harness / multi-agent skills
- adapters for Claude Code, Codex CLI, Gemini CLI, Cursor, and Windsurf

It is open source and local-first: markdown skills, examples, adapters, and docs.

I would love feedback from builders using AI coding tools every day. Which workflow should become a skill next?
```
