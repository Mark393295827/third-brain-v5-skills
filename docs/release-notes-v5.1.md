# Third Brain V5.1 Release Notes

Release date: 2026-05-14

## Summary

V5.1 packages the repository as a more complete Third Brain system: a visible architecture diagram, a 17-skill catalog, stronger skill templates, richer examples, and clearer adapter routing for Claude, Codex, Gemini, Cursor, and Windsurf.

## Highlights

- Added the README-top architecture infographic for the full LLM + Skills + Obsidian closed loop.
- Added `agentic-engineering` for refactoring skills and workflows into model-native execution loops.
- Expanded `wiki-ingest`, `cognitive-compile`, and `creativity-engine` around the V5 source-to-behavior-to-creativity loop.
- Standardized quality gates across core engineering and evaluation skills.
- Expanded Cursor and Windsurf skill routing coverage.
- Added seed wiki/system content for daily OKR, news signal triage, and output templates.
